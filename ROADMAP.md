# The Forgotten — Roadmap

What's done, what's next, what's dreamed about.

## ✅ Done

- **Palestone blocks** — palestone, polished palestone, palestone bricks (textures, models, loot tables, mining tags)
- **Portal system** — reinforced deepslate frame + echo shard activation, bidirectional teleportation, frame validation
- **Portal visuals** — animated texture (32-frame spectral wisps), sculk soul + enchant particles, ambient sculk catalyst sound
- **Noise-based worldgen** — nether-style caverns, 192 blocks tall, palestone default block
- **Surface rules** — polished palestone floors, palestone brick ceilings, deepslate patches
- **Biome: forgotten_wastes** — sculk veins, sculk patches, glow lichen, soul sand valley ambient loop
- **Dimension type** — no skylight, has ceiling, 0.05 ambient light, end sky effects

## 🔨 Next Up

- [ ] **Fix portal landing** — currently lands on bedrock roof instead of inside caverns. Search from y=150 down, or find largest air pocket

- [ ] **Mob/entity teleportation** — currently player-only, items and mobs should pass through too
